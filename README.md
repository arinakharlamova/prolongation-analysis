> **Аналитический отчёт по эффективности пролонгации договоров аккаунт-менеджерами за 2023 год**

---

## Цель проекта

Расчёт и анализ коэффициентов пролонгации (K1, K2) для оценки эффективности работы аккаунт-менеджеров на основе данных о завершённых проектах и финансовых отгрузках.

### Задачи:
- Рассчитать коэффициенты пролонгации для каждого менеджера и отдела в целом
- Проанализировать динамику за каждый месяц 2023 года
- Подготовить итоговые значения за год (средневзвешенные)
- Сформировать аналитический отчёт для руководства

---

## Методология

### Коэффициент K1 — Пролонгация в первый месяц

$$K1_M = \frac{\sum\text{Отгрузка}_{M}}{\sum\text{Отгрузка}_{M-1}}$$

**Для проектов, у которых:**
- Последний месяц реализации = M-1
- Есть отгрузка в месяце M

### Коэффициент K2 — Пролонгация во второй месяц

$$K2_M = \frac{\sum\text{Отгрузка}_{M}}{\sum\text{Отгрузка}_{M-2}}$$

**Для проектов, у которых:**
- Последний месяц реализации = M-2
- НЕТ отгрузки в месяце M-1
- ЕСТЬ отгрузка в месяце M
